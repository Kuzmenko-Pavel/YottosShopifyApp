HOW TO CREATE A PIXEL IN THE BUSINESS MANAGER OF THE FACEBOOK

STEP 1

To create a new pixel for facebook, you will need to have admin rights in your Business Manager account.

If you do have the rights, you will need to go to Campagin Settings in Business Manager

If there are more then one campaign in your account - select the one you need.

{% include "include/faq_item_more_link.html" %}

STEP 2

After that you will have to select 'Data Sources' in the menu to the left.

{% include "include/faq_item_more_link.html" %}

STEP 3

You will need to select 'Pixels' option in this field of this tab.

{% include "include/faq_item_more_link.html" %}

STEP 4

In the opened section you can see all the created pixels, and you can create new one

To make a new pixel you will need to click on the 'Add' button

{% include "include/faq_item_more_link.html" %}

STEP 5

In the new window you will need to select the name of the pixel.

You also may fill in the URL of the site, which will use this exact pixel (but this is optional).

{% include "include/faq_item_more_link.html" %}

STEP 6

When you have finished filling in pixel name field with or without site's URL, click 'Continue'

{% include "include/faq_item_more_link.html" %}

STEP 7

After this Facebook will suggest you to install the Pixel on your site, or return to campaign management

{% include "include/faq_item_more_link.html" %}

STEP 8

Select the 'Return to campaign management' option.

{% include "include/faq_item_more_link.html" %}

STEP 9

Pixel is created. You can see the new pixel in the menu to the left.

{% include "include/faq_item_more_link.html" %}

STEP 10

To send and install Facebook Pixel on your site - open your pixel by clicking on 'Open in Events Manager'

{% include "include/faq_item_more_link.html" %}

STEP 11

Click on the 'Settings' button

{% include "include/faq_item_more_link.html" %}

STEP 12

Click on the 'Install pixel' button in the dropdown list, and select one of the options of pixel installation.

{% include "include/faq_item_more_link.html" %}

STEP 13

After this, the only thing left for you to do is to install the pixel's code on your site according to Facebook's reccomendation.

{% include "include/faq_item_more_link.html" %}
